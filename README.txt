Setup:
       1: install pip using 'apt install python-pip'
Run The Following Commands:
       2: pip install virtualenv
       3: virtualenv flask
       4: mkdir tmp
       5: ./setup.txt (you may need to change permissions)
       6: source flask/bin/activate (this starts the enviornment)
       7: ./db_create.py
       8: ./db_migrate.py
       9: ./run.py to run the app
